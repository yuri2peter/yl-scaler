# yl-scaler

Make your browser's solution looks just like what you want!

> It's not magic.

# Quick Start

```
ylScaler.quickStart(1920);
```

# Advanced

```
window.onresize = ylScaler([
  { stage: 'md', value: 1440, min: 1024, max: 1440 },
  { stage: 'lg', value: 1920, min: 1441, max: 1920 },
])
```
